参考：openmpi在centos上的安装

  • 下载openmpi-1.8.8.tar.gz
  • 上传openmpi-1.8.8.tar.gz到服务器指定目录，例如：/home/software

  • 解压

  • 同级目录创建安装目录openmpi_install

  • 进入解压后的目录openmpi-1.8.8，在该目录下配置安装路径

  • 执行make

  • 执行make install

  • 在用户目录下设置mpi的路径
